# 企业产品知识库

## 产品概述
本企业提供一站式智能客服解决方案，覆盖在线客服、工单管理、CRM客户管理、订单查询等核心功能。

## 常见问题 FAQ

### 如何重置密码？
用户可在登录页面点击"忘记密码"，通过绑定邮箱或手机号获取验证码后重置密码。若无法自助重置，请联系企业管理员。

### 订单发货时间
普通订单在支付成功后 24 小时内发货，定制产品需要 3-5 个工作日。您可以通过订单查询功能查看实时物流状态。

### 退款政策
7 天内无理由退货（定制产品除外），退款将在收到退货后 3-5 个工作日原路退回。建议先联系客服确认退货流程。

### 系统故障报修
如遇系统无法访问、功能异常等问题，请先尝试清除浏览器缓存或更换浏览器。若问题持续，请提供截图通过工单系统提交。

### VIP 客户权益
VIP 客户享有专属客服通道、优先工单处理、季度产品使用报告等权益。如需了解您的客户等级，可使用 CRM 查询功能。

## API 接入指南

### 获取 API Key
登录管理后台 -> 设置 -> API 管理 -> 生成 Key。请注意保管，不要泄露给他人。

### 接口限流
免费版：100 次/分钟，企业版：1000 次/分钟，旗舰版：无限制。
